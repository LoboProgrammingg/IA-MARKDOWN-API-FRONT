Central de Dúvidas

Perguntas Frequentes

Que tipo de documentos a IA consegue analisar?

Nossa IA é treinada para processar uma vasta gama de formatos, incluindo PDFs, documentos do Word, planilhas do Excel e até mesmo imagens de documentos digitalizados. Ela é otimizada para entender a estrutura de documentos governamentais como licitações, contratos, portarias e relatórios.

A plataforma é segura para dados governamentais sensíveis?

Absolutamente. A segurança é nossa prioridade máxima. Utilizamos criptografia de ponta-a-ponta, armazenamento em nuvem governamental certificada e seguimos rigorosos protocolos de controle de acesso baseados em perfis de usuário, garantindo total conformidade com a LGPD e outras regulamentações.

É necessário treinamento técnico para usar a plataforma?

Não. A Interface Synapse foi projetada para ser intuitiva. Se você sabe usar um chat ou um buscador, você saberá usar nossa plataforma. Oferecemos um onboarding completo e materiais de suporte para garantir que sua equipe aproveite ao máximo todas as funcionalidades desde o primeiro dia.
